# 🍽️ Restaurant & Cafe Management System

An ASP.NET MVC 5 based Restaurant & Cafe Management System developed to simplify restaurant operations such as menu management, order tracking and customer management.

---

## 📌 Project Overview

This project was developed as a web-based management system for restaurants and cafes. It provides an easy-to-use interface for managing daily business operations efficiently.

The application allows administrators to manage menus, monitor orders, organize customer information and perform various administrative tasks through a centralized dashboard.

---

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/irfancotoglu/RestaurantCafeManagementSystem.git
```

### 2. Open the solution

Open the project using **Visual Studio 2022** (or Visual Studio 2019).

### 3. Restore NuGet Packages

Visual Studio will automatically restore missing packages.

### 4. Database

Restore the SQL Server backup file:

```
Database/lokanta_cafe_takip.bak
```

Update the **connection string** inside:

```
Web.config
```

according to your SQL Server instance.

### 5. Run

Build the solution and start the project.
